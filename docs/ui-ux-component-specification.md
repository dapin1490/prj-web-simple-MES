심플 MES(Simplified MES) 프로젝트의 프론트엔드 구현을 위한 UI/UX 컴포넌트 명세서입니다. 본 문서는 Vue.js 3.x 스택을 기반으로 하며, 사용자 플로우 및 API 상세 명세서를 시각적으로 구현하기 위한 가이드를 제공합니다.

## 심플 MES UI/UX 컴포넌트 명세서 (v1.0)

### 1. 디자인 원칙 및 시스템 레이아웃

- 디자인 원칙: 현장 작업자의 시인성을 최우선으로 하며, 상태 변화(정상/주의/위험)를 색상(Green/Yellow/Red)으로 직관적으로 표현함.
- 공통 레이아웃:
    - Sidebar: 모듈별 메뉴 이동 (대시보드, 수주 관리, 공정 모니터링, 품질 리포트).
    - Header: 현재 로그인 사용자 권한(Admin/Operator) 및 시뮬레이션 상태 표시.
    - Content: 각 모듈의 핵심 컴포넌트 배치.

### 2. 주요 화면별 컴포넌트 상세

### 2.1. 메인 대시보드 (Executive Summary)

공장 전체의 가동 현황을 한눈에 파악하는 화면입니다.

- KPI 카드 컴포넌트:
    - 목표 대비 실적(Progress): 일일 총 주문량 대비 생산 완료량 시각화.
    - 품질 합격률(Pass Rate): 전체 로트 중 `pass_fail`이 `true`인 비율 표시.
- 현장 현황판: 현재 가동 중인 설비별 'LOT번호' 및 실시간 진척도($P$) 게이지 차트.
    - $P = \frac{t_{current}}{t_{total}} \times 100$ 공식 적용.

### 2.2. 수주 및 생산 계획 (Order & Planning)

전처리된 수주 데이터를 확인하고 작업을 할당하는 화면입니다.

- 수주 현황 테이블: `SalesOrders` 데이터를 로드하여 제품명, 주문량, 납기일자를 리스트업함.
- 생산 트리거 위젯: `current_inven`이 `safety_stock` 이하인 품목을 강조하여 강조 표시함.
- 작업 지시 발행 버튼: 수주 항목 선택 시 해당 사양(`planned_qty`, `machine_id`)에 맞는 `WorkOrder`를 생성하는 모달(Modal) 실행.

### 2.3. 실시간 공정 모니터링 (Execution Monitoring)

시뮬레이터에서 전송되는 데이터를 실시간으로 시각화하는 핵심 화면입니다.

- 실시간 트렌드 차트: WebSocket(STOMP)으로 수신되는 `temp_pv`(실측 온도), `temp_sp`(지시 온도), `cr_temp`(목표 온도) 및 `speed`(포속)를 실시간 꺾은선 그래프로 출력함. 지시·목표는 참고선 또는 별도 시리즈로 표현할 수 있다.
    - Y축: 온도(℃), 속도(m/min) / X축: 시간(Timestamp).
- 로그 스트리밍 패널: 서버에서 삽입되는 `ProductionLogs` 레코드를 텍스트 로그 형태로 실시간 롤링 업데이트.

### 2.4. 품질 검사 및 리포트 (Quality & Reporting)

생산 완료된 데이터의 통계와 보고서를 관리하는 화면입니다.

- 품질 분포 산점도(Scatter Plot): 로트별 `color_de` 값을 좌표로 표시하여 품질 편차 시각화.
- 검사 성적서 뷰어: 특정 로트의 작업 사양, 생산 로그, 품질 판정 결과가 결합된 표준 양식 프리뷰.
- PDF 내보내기 버튼: 뷰어에 표시된 내용을 문서 파일로 저장하는 기능 제공.

### 3. 주요 UX 인터랙션 설계

| 이벤트 | 동작 (Action) | 결과 (Reaction) |
| --- | --- | --- |
| 시뮬레이션 시작 | `Simulation Start` 버튼 클릭 | 시뮬레이터 가동 및 대시보드 차트 실시간 갱신 시작. |
| 품질 이상 발생 | `color_de` $\ge$ 1.0 감지 시 | 해당 설비 카드 배경색 Red 점멸 및 시스템 알림 발생. |
| 설비 이상(공정) | 백엔드가 `temp_sp`·`temp_pv`·`speed` 등으로 이상 판정 시 (`docs/api-details.md` §6.1) | 토스트·배너 등으로 메시지 표시. 품질 알림과 구분 가능한 문구·색상 권장. |
| 진척도 100% 도달 | 특정 로트 생산 완료 시 | 작업 상태가 '완료'로 자동 전환되며 품질 검사 데이터 매핑. |
